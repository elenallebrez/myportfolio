Juego estilo Reigns con humor sevillano y decisiones absurdas

Python Pygame JSON

Ver en GitHub Demo online

Detalles técnicos

  • Stack: Python, JSON
  • Arquitectura: Pygame modular (carpetas core, screens, ui, data)
  • Mi rol: Developer & Game Designer
  • Estado: Publicado

Cómo lo construí

  1. Diseñé la estructura modular para separar pantallas, lógica y datos.
  2. Implementé animaciones de swipe izquierda/derecha para decisiones al estilo Reigns.
  3. Integré un sistema de condiciones de victoria/derrota, con mensajes de final personalizados y cargados desde JSON.
  4. Mejoré el frontend con elementos visuales: contenedor de stats con azulejos, interfaz más limpia y humor local.

Resultados & aprendizajes

  • Resultados: Proyecto jugable y publicado en GitHub; feedback positivo por la originalidad cultural y el humor absurdo.
  • Aprendizajes: Profundicé en Pygame, diseño de arquitecturas modulares y uso de JSON para narrativa dinámica; también experimenté con UX orientado a decisiones rápidas.
  • Siguientes pasos: Añadir más eventos y personajes, mejorar efectos visuales/sonoros, e integrar un sistema de logros para aumentar rejugabilidad.
