VRP solved by clustering and genetic algorithm

Application for solving the Vehicle Routing Problem using clustering and genetic algorithms to optimize distribution routes.

Python Scikit-learn NumPy Matplotlib

View on GitHub

The Vehicle Routing Problem (VRP) is a classic logistics optimization challenge. To tackle it, I combined a clustering algorithm to group nearby customers and a genetic algorithm to optimize routes within each cluster. This approach reduced total travel distance, improved fleet utilization, and provided a clear framework for analyzing route efficiency.

Technical Details

  • Stack: Python, Scikit-learn, NumPy, Matplotlib
  • Architecture: Modular script with separate modules for clustering, fitness evaluation, crossover, mutation, and visualization
  • My role: Algorithm design, implementation, and visualization of results
  • Duration: 3 weeks (2024)
  • Status: Functional MVP
