VRP solved by clustering and genetic algorithm

Aplicación para resolver el Vehicle Routing Problem combinando técnicas de clustering y algoritmos genéticos, orientado a optimizar rutas de distribución.

Python Scikit-learn NumPy Matplotlib Heuristics

Ver en GitHub

Resumen

El Vehicle Routing Problem (VRP) es un desafío clásico de optimización en logística y distribución. Para resolverlo, combiné un algoritmo de clustering para agrupar clientes y un algoritmo genético para optimizar las rutas dentro de cada cluster. El resultado fue una reducción significativa en la distancia total recorrida y un mejor aprovechamiento de la flota de vehículos.

Detalles técnicos

  • Stack: Python, Scikit-learn, NumPy, Matplotlib
  • Arquitectura: Script modular con módulos para clustering, fitness, crossover y visualización
  • Mi rol: Diseño del enfoque, implementación de algoritmos y visualización de resultados
  • Tiempo: 3 semanas (2024)
  • Estado: MVP funcional

Cómo lo construí

  1. Definí el dataset de clientes y la función de distancia para modelar el problema.
  2. Implementé clustering (K-means) para reducir la complejidad dividiendo clientes por proximidad.
  3. Diseñé un algoritmo genético con operadores de selección, cruce y mutación para encontrar rutas eficientes dentro de cada cluster.
  4. Visualicé las rutas resultantes para analizar mejoras respecto a soluciones iniciales.

Resultados & aprendizajes

  • Aprendizajes: Profundicé en algoritmos evolutivos, heurísticas de clustering y técnicas de optimización combinatoria.
