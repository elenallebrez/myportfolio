Portfolio Web Personal

Mi portfolio personal construido con HTML y CSS, mostrando mis proyectos, experiencia, habilidades y contacto profesional.

HTML CSS Responsive Design Portfolio

Ver en GitHub

Esta web sirve como portfolio profesional, organizada en secciones de “Sobre mí”, “Proyectos” y “Contacto”. Está diseñada para ser clara, responsive y fácil de navegar, destacando habilidades técnicas y experiencias relevantes.

Technical Details

  • Stack: HTML, CSS, Responsive Design
  • Arquitectura: Estructura modular con secciones semánticas y contenedores reutilizables
  • Mi rol: Diseño de la interfaz, maquetación y estilizado del portfolio completo
  • Tiempo: 2 semanas (2025)
  • Estado: Web funcional y responsive
Cómo lo construí
  1. Diseñé la estructura HTML semántica, usando secciones y contenedores consistentes.
  2. Estilicé el sitio con CSS, utilizando variables, grids y flexbox para mantener consistencia y responsividad.
  3. Optimicé la navegación y la experiencia de usuario en desktop y móvil.
  4. Incluí enlaces a GitHub y contacto profesional, destacando proyectos y habilidades.
