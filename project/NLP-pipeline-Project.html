NLP Pipeline

End-to-end pipeline to process, enrich, and adapt chatbot conversations for a mental health application.

Python OpenAI Gemini NumPy Data Cleaning Embeddings Cosine Similarity

View on GitHub

During my professional experience, I was tasked with adapting a mental health chatbot to different cultural contexts. To address this, I designed an NLP pipeline capable of cleaning raw data, extracting conversational context, and generating embeddings with multiple LLMs. This work delivered production-ready datasets for training, a comparative analysis of OpenAI and Gemini embeddings, and the generation of synthetic “red flag” phrases for moderation and cultural adaptation.

Technical Details

  • Stack: Python, NumPy, OpenAI API, Gemini API
  • Architecture: Modular Jupyter notebooks covering cleaning, embeddings, data generation, and analysis
  • My role: Designed and implemented the pipeline — data preprocessing, embeddings generation, cross-model comparison, and synthetic dataset creation for moderation
  • Timeline: August–September 2025 (India)
  • Status: Deployed and published

How I built it

  1. Standardized and cleaned raw Excel conversations into structured JSON datasets.
  2. Implemented embeddings generation with both OpenAI and Gemini, developing a cosine similarity–based comparison tool.
  3. Created a synthetic dataset generator to produce culturally sensitive “red flag” phrases for moderation testing.
