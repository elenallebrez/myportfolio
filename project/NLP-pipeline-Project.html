NLP pipeline

Pipeline de NLP para procesar y enriquecer conversaciones de un chatbot de salud mental.

Python OpenAI Gemini NumPy Data Cleaning Embeddings Cosine Similarity

Ver en GitHub

El proyecto nace de la necesidad de adaptar un chatbot de salud mental a diferentes contextos culturales. Para ello, diseñé un pipeline de NLP capaz de limpiar datos, extraer contexto y generar embeddings con diferentes LLMs. Como resultado, conseguimos datasets listos para entrenamiento, análisis comparativo de embeddings entre OpenAI y Gemini, y generación sintética de frases sensibles para tareas de moderación.

Technical Details

  • Stack: Python, NumPy, OpenAI API, Gemini API
  • Arquitectura: Jupyter Notebooks organizados en módulos de limpieza, embeddings, generación de datos y análisis
  • Mi rol: Diseño del pipeline, procesamiento de datos en bruto, generación de embeddings, comparación entre modelos y síntesis de datasets para moderación cultural
  • Tiempo: Agosto–Septiembre 2025 (India)
  • Estado: Publicado
Cómo lo construí
  1. Estandaricé y limpié los datos crudos en Excel → JSON mediante varios pasos de preprocesamiento.
  2. Implementé la generación de embeddings en paralelo con OpenAI y Gemini, y desarrollé un comparador con cosine similarity.
  3. Diseñé un generador de datasets sintéticos de “red flags” para pruebas de moderación y adaptación cultural.
