Sevilla Reigns

Decision-based game with Sevillian humor, inspired by Reigns.

Python Pygame JSON

View on GitHub

This project was a fun side project inspired by the game Reigns. I designed a small, interactive game set in Sevilla, featuring local humor and absurd scenarios. Players make decisions by swiping left or right, and each choice affects the game state, such as popularity, wealth, or relationships. The game showcases a modular architecture, visual enhancements, and dynamic outcomes, providing both entertainment and a learning experience in game development and UI design.

Technical Details

  • Stack: Python, JSON
  • Architecture: Modular Pygame structure (folders core, screens, ui, data)
  • My role: Developer & Game Designer
  • Status: Published
How I built it
  1. Designed a modular structure separating screens, game logic, and data.
  2. Implemented left/right swipe animations for decision-making in the style of Reigns.
  3. Developed a system for win/lose conditions with personalized end messages loaded from JSON.
  4. Enhanced the UI with visual elements: stats container with tiles, cleaner interface, and local humor integration.
