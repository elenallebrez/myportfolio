Food Delivery System (CRUD + API REST)

Plataforma de pedidos de comida entre restaurantes y clientes

Node.js Express MariaDB React Native REST API

Ver en GitHub

Los restaurantes necesitan una plataforma digital para gestionar pedidos y llegar a más clientes. Por eso diseñamos un sistema completo de backend y apps móviles para clientes y dueños de restaurantes.. El resultado, una app funcional estilo Glovo/UberEats, con gestión de pedidos, productos, restaurantes y usuarios.

Technical Details

  • Stack: Node.js, Express, MariaDB, React Native
  • Arquitectura: Cliente-servidor con API REST; apps móviles separadas para clientes y dueños
  • Mi rol: Desarrollo de backend para pedidos de clientes y frontend de la app de cliente
  • Tiempo: 4 meses, 2025
  • Estado: Proyecto académico (MVP funcional)
