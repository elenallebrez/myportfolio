Python Term Project

Academic Python project analyzing MBA student data and employability.

Python Namedtuples Data Visualization

View on GitHub

The challenge was to work with a real MBA student dataset and extract insights on grades and employability. The solution involved building a Python system using namedtuples, filters, statistics, and visualizations to explore and analyze the data. The result was a robust academic project that allowed a thorough examination of key factors affecting MBA student employability.

Technical Details

  • Stack: Python, Namedtuples, Matplotlib, Unit Testing
  • Architecture: Independent modules (analysis, tests, visualization) + CSV dataset
  • My role: Full development of the project, from data preprocessing to analysis and visualization
  • Timeline: Semester 2022
  • Status: Completed and submitted

How I built it

  1. Chose a relevant dataset of MBA students and explored its structure and variables.
  2. Implemented Python namedtuples to store structured student data efficiently.
  3. Developed filtering and analysis functions to extract key insights on grades and employability factors.
  4. Created visualizations using Matplotlib to present findings clearly and intuitively.
  5. Performed unit testing to ensure accuracy of analysis functions and correctness of results.
