Python Term Project

Proyecto académico en Python para analizar datos de estudiantes de MBA y su empleabilidad.

Python Namedtuples Gráficos

Ver en GitHub

El problema consistía en trabajar con un dataset real de estudiantes y extraer conclusiones sobre sus calificaciones y empleabilidad. La solución fue implementar un sistema en Python que utiliza namedtuples, filtros, estadísticas y gráficos para explorar los datos. El resultado fue un proyecto sólido que permite analizar factores clave en la empleabilidad de estudiantes de MBA.

Technical Details

  • Stack: Python, Namedtuples, Matplotlib, Tests unitarios
  • Arquitectura: Módulos independientes (análisis, tests, gráficos) + dataset CSV
  • Mi rol: Desarrollo completo del proyecto, desde el preprocesado de datos hasta la visualización
  • Tiempo: Cuatrimestre 2022
  • Estado: Proyecto finalizado y entregado
