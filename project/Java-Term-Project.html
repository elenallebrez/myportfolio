Java Term Project

Proyecto académico en Java para procesar y analizar datos de béisbol, desarrollado en primero de carrera.

Java Collections Streams

Ver en GitHub

El problema era diseñar un sistema que pudiera trabajar con un dataset real de béisbol y aplicar conceptos de programación orientada a objetos. La solución fue implementar clases, colecciones y fábricas para modelar lanzamientos y partidos, además de operaciones con streams para consultas. Como resultado, logré un proyecto funcional y bien estructurado que me permitió afianzar mis conocimientos en Java y manejo de datos.

Technical Details

  • Stack: Java
  • Arquitectura: Clases de dominio + fábrica de objetos + contenedor de colecciones
  • Mi rol:Desarrollo completo del proyecto, modelado de datos y lógica de operaciones
  • Tiempo: 4 meses, 2023
  • Estado: Proyecto finalizado y entregado
